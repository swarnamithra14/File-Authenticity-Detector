# File Authenticity Detector

A web-based file authenticity detection system that analyzes file metadata and content to determine whether files are original or tampered with.

## About

This project detects suspicious files using a combination of metadata inspection, file content analysis, and scoring heuristics. It is designed to help users verify file authenticity and identify potentially altered or fraudulent documents.

## Key Components

- `backend/`
  - `app.py`: Flask application entry point.
  - `hidden_detector.py`: Hidden feature detector logic.
  - `metadata.py`: File metadata extraction utilities.
  - `report_generator.py`: Generates report output for analysis results.
  - `scorer.py`: Computes authenticity score based on file attributes.
  - `spam_detector.py`: Detects spam or suspicious files.
  - `utils.py`: Shared helper functions.
- `templates/`: HTML templates for upload, result, and spam pages.
- `static/`: CSS styling for the web interface.
- `assembly/`: Assembly files and test artifacts used for demonstration.

## Usage

1. Run the Flask app from `backend/app.py`.
2. Upload a file via the web interface.
3. Review the authenticity report generated by the application.

## Deployment

This project can be deployed on platforms like Netlify or a cloud VM by serving the backend and frontend together.
